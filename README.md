96pxgames
=========

####Dependencies

* pygame
* PyOpenGL

####Getting Started

execute ```python simulator/simulator.py``` to start the Simulator (windows only)

See sample.py for an example implementation.


####Known Bugs

* Simulator window will not respond to UI events, still works though
